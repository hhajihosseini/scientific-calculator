# Scientific Calculator

A scientific calculator built with **React**. This project allows users to perform basic arithmetic calculations as well as more advanced scientific operations like trigonometric functions, logarithms, and exponentiation.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, and division.
- **Scientific Operations**: `sin`, `cos`, `tan`, `log`, `ln`, `sqrt`, `x^y`, and more.
- **Constants**: Supports mathematical constants such as `Ï€` and `e`.
- **Responsive Design**: Works well on both desktop and mobile devices.

## Installation

To run the project locally on your machine, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/hhajihosseini/scientific-calculator.git
   cd scientific-calculator
2. **Install dependencies**:

 bash
npm install

3. **Start the development**:

bash
npm start

The application will open in your browser at http://localhost:3000.

## usage
1. Enter numbers using the calculator buttons
2. Click on operations (sin, cos, log, etc.) for scientific functions
3. Press = to calculate the result
4. Use C to clear the display
